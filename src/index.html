Build System Template

I have created a build system "boilerplate". I'm in the process of writing the documentation and current usage for each item and the pacakges associated.

What you see below are the base elements of the build.

git | version control system

nodejs | runtime environment

npm | node package manager

gulp.js | build tool / task runner

bower | front-end package manager

Browsersync | cross-browser testing & live reload

Bootstrap | front-end framework
